## üõ† Installation

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/magic-mirror-pi.git
cd magic-mirror-pi
```

### 2. Run the App
To view the app in your browser:
```bash
# Assuming you are using a standard dev server
npm install
npm run dev
```

### 3. Run in Kiosk Mode (Raspberry Pi)
To launch directly from the terminal in full-screen (replacing `localhost` with your actual port):
```bash
chromium-browser --noerrdialogs --disable-infobars --kiosk http://localhost:5173
```
